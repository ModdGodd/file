Corbert

Comes in two varieties; one downside is that I got the fonts from different sources, so it looks different when compared to each other (unlike the next one).

# Type font-family Value Description
1 Regular/Medium 'Corbert' Perfect UI font, its too modern to be a body typeface, but it is just behind Monteserrat, which is probably by now too overused.
2 Bolded 'Corbert B' Its a great font if its your style, I reccomend to set everything to it; the only catch was that font-weight and bold tags worked bad with the normal Corbert, so I had to dig out a bold version manually
